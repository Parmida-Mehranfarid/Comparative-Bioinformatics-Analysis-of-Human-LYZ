# Data

This folder contains the nucleotide and protein sequences used throughout this bioinformatics project.

## Files

- Human_LYZ_mRNA.fasta — Human LYZ mRNA sequence retrieved from the NCBI Nucleotide database.
- Human_LYZ_Protein.fasta — Human LYZ protein sequence retrieved from the NCBI Protein database.

These sequences were used as the primary input for BLAST analysis, multiple sequence alignment, phylogenetic analysis, protein characterization, and primer design.
